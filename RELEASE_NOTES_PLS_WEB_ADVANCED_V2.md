# Release notes — CanchariOS PLS Web Advanced V2

- Suite avanzada PLS-SEM expuesta en la interfaz web.
- Configuración completa de Q², PLS-Predict, HTMT-CI, SRMR, CMB, VAF, IPMA, cópula, MICOM y MGA.
- Variables de control incorporadas de forma explícita.
- FIMIX-PLS agregado mediante `seminrExtras`.
- Segmentos FIMIX utilizables como fuente para MICOM/MGA.
- Comparación de modelos directo, paralelo y secuencial.
- Resultados y Word ampliados.
- Dos pruebas nuevas; nueva meta local: 16 PASS en certificación y 37 PASS consolidados.
